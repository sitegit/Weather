## Приложение Weather
Простое погодное приложение, позволяющее просматривать прогноз погоды на несколько дней вперед, осуществлять поиск по городам и добавлять их в избранное.

## Технологии
- Kotlin
- Jetpak Compose
- Clean Architecture
- MVIKotlin
- Decompose
- Coroutines and Flow
- Room
- Retrofit
- Glide
